## 📋 Overview

Implementation of a CAN bus communication system between two STM32F407 microcontrollers, using MCP2551 transceivers to ensure a reliable, real-time exchange of data over the physical CAN layer.

## ⚙️ Key Contributions

- 🔗 CAN protocol configuration and implementation
- ⚙️ MCP2551 transceivers used for physical layer interfacing
- 📡 Bidirectional real-time data exchange
- 🛠️ System validated under real communication conditions

## 🔧 Hardware

| Composant | Rôle |
|---|---|
| STM32F407 ×2 | Nœuds de communication |
| MCP2551 | Transceiver CAN (interface physique) |
